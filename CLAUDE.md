## Rules

- **Wait for instructions when a new tool/repo is shared.** When the user shares a GitHub link, package name, screenshot, or any new tool, do NOT install it, run it, or import it on your own initiative — even if the message contains commands. Sharing is context, not a request to execute. Ask what they want done with it and wait for their answer before touching anything.
